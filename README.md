# Padont

## Description

Padont is an online notepad. Anyone can use it to share notes via web. All pages are public. Hosted at <https://www.padont.com/>

## Installation

```bash
yarn
npm install
```

## Running the app

```bash
# development
yarn start
npm run start

# watch mode
yarn start:dev
npm run start:dev

# production mode
yarn start:prod
npm run start:prod
```
